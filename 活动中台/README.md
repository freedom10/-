### 功能
支持多种抽奖、买赠、签到、等多种活动模版，丰富会员运营能力。
支持中奖实时短信通知，支持奖品库存阀值预警（需要接入短信网关）
支持实物，虚拟，微信红包，等多种奖品
支持前端H5模版化配置，模版化活动替换素材快速上线
支持领奖关注公众号，增加公众号引流
支持第三方订单流水增加用户权益（需要接入第三方订单流水）

### 技术方案
系统架构模块化设计，松耦合，活动实现灵活配置。
分布式服务与存储，支持高并发活动场景。
